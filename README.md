# Crypto 测试项目


一个模拟数字货币交易平台的实战测试项目，用于学习和展示金融业务测试的核心技能。

## 🎯 项目目标
通过一个完整的测试周期，演示如何对数字货币交易业务进行质量保障，涵盖：
*   **功能测试**与用例设计
*   **API接口测试**与自动化断言
*   **测试计划**与报告编写

## 📁 项目结构
futu-crypto-test-project/
├── day1_hello.py # 环境验证
├── day1_portfolio.py # 资产组合模拟器（核心数据结构）
├── day2_api.py # 加密货币实时价格API调用（含异常处理）
├── README.md # 本项目说明文件
└── .gitignore


## 🚀 快速开始
1.  克隆项目：`git clone https://github.com/Carlfifi/crypto-test-project.git`
2.  安装依赖：`pip install requests`
3.  运行示例：`python day2_api.py` 即可获取实时加密货币价格。

*   **工具**: PyCharm, Git, Postman
